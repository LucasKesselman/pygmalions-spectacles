# AR Web App

A lightweight AR experience built with Zappar and Babylon.js, designed for users to store and view their own AR designs through Firebase. This project works seamlessly in vscode.dev and supports collaborative development.

## Features

Zappar augmented reality  
Babylon.js 3D rendering  
User specific AR design storage via Firebase  
Support for 3D models, textures, and config files  
Docker and Dev Container ready
